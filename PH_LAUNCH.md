## Description

Indie Updates Hub is the AI-powered update distribution tool built for indie makers who are tired of copy-pasting the same changelog across 5 platforms.

Write your update once in Markdown, then auto-generate:
- 🐦 X/Twitter threads (optimized for engagement, under 280 chars)
- 📋 Embeddable changelog pages (dark / minimal / gradient themes)
- 💬 Discord & Slack notifications
- 📧 Newsletter copy

**Why we built it:**
We noticed indie makers spending 20+ minutes formatting the same update for different platforms. That's 20 minutes not spent on the product. Indie Updates Hub cuts that to 2 minutes.

**Key features:**
- ✍️ Markdown editor with live preview
- 🤖 AI thread writer (auto-splits your update into tweet-sized chunks)
- 🎨 3 embeddable changelog themes
- 🔗 One-click publish to 5 channels
- 🏷️ Custom domain support (Creator plan)

**Pricing:** Free forever for 3 updates/mo. Creator plan at $15/mo.

Built in public. Ship faster.
